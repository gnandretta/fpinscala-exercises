## Usage

Run the Scala REPL, load some file or paste some code, and then invoke some function.

```
$ scala
scala> :load chapter2.scala
scala> MyModule.fib(5)
scala> :paste
# copy and paste code from chapter3.scala
scala> List.tail(List(1,2,3,4))
```
